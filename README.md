# MarvinARM
![Marvin](images/marvin.jpeg)

## Requirement
```
ROS Melodic
Azure Kinect SDK
Openmanipulator X
```
## Installation
```
cd ~/catkin_ws
git clone https://github.com/thedavekwon/MarvinARM.git
catkin_make
```
## Usage
```
roslaunch mimic mimic.launch
```
